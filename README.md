# rustfm

A fast and lightweight Rust file manager with an integrated terminal. Manage files and execute commands all in one place.

## Features

- **File Management**: Browse, create, copy, move, and delete files and directories
- - **Integrated Terminal**: Built-in terminal for quick command execution
  - - **Fast Performance**: Written in Rust for speed and reliability
    - - **Cross-platform**: Works on multiple operating systems
     
      - ## Installation
     
      - Build from source:
     
      - ```bash
        git clone https://github.com/Divhanthelion/rustfm.git
        cd rustfm
        cargo build --release
        ```

        The compiled binary will be in `target/release/`

        ## Usage

        Run rustfm:

        ```bash
        ./target/release/rustfm
        ```

        Navigate files using the keyboard and interact with the integrated terminal.
